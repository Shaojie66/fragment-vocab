# 碎片时间背单词工具 UI/UX 设计方案

## 1. 设计原则

### 1.1 核心理念
- **极简主义**：减少视觉干扰，聚焦单词本身
- **快速响应**：3秒内完成一次学习
- **非侵入式**：融入工作流，不打断思路
- **即时反馈**：操作后立即给予视觉确认

### 1.2 设计约束
- 单词卡展示时间：10-12秒
- 卡片尺寸：固定，不可调整
- 交互方式：鼠标点击 + 快捷键
- 动画时长：≤ 300ms

### 3.5 交互状态
1. **默认状态**：卡片从右上角淡入（fade + slide）
2. **悬停状态**：按钮背景色加深，鼠标变为pointer
3. **点击状态**：按钮缩放0.95，持续100ms
4. **消失状态**：卡片淡出（fade），持续200ms

### 3.6 快捷键提示
在卡片底部显示快捷键提示（可选）：
```
Cmd+K 认识  |  Cmd+J 不认识  |  Esc 跳过
```
- Font-size: 12px
- Color: #BDC3C7
- Position: 距离底部12px

## 4. 菜单栏图标设计

### 4.1 图标风格
- 类型：线性图标（Line Icon）
- 尺寸：22x22px（@1x），44x44px（@2x）
- 线宽：2px
- 颜色：系统默认（自适应深色/浅色模式）

### 4.2 图标方案

#### 方案A：书本+闪电
```
象征：快速学习
元素：打开的书本 + 闪电符号
适用场景：强调速度和效率
```

#### 方案B：单词卡片
```
象征：单词卡
元素：矩形卡片 + 字母A
适用场景：直观表达产品功能
```

#### 方案C：脑袋+单词
```
象征：记忆
元素：简化的头部轮廓 + 单词符号
适用场景：强调记忆和学习
```

### 4.3 推荐方案
**采用方案B（单词卡片）**
- 最直观表达产品功能
- 图标识别度高
- 适合菜单栏小尺寸展示

### 4.4 状态图标
```
正常状态：黑色/白色（跟随系统）
暂停状态：灰色 + 暂停符号叠加
学习中：蓝色 + 微动画（可选）
```

## 7. 动画设计

### 7.1 卡片出现动画
```css
@keyframes cardEnter {
  from {
    opacity: 0;
    transform: translateX(20px) translateY(-20px);
  }
  to {
    opacity: 1;
    transform: translateX(0) translateY(0);
  }
}

duration: 300ms
easing: cubic-bezier(0.4, 0, 0.2, 1)
```

### 7.2 卡片消失动画
```css
@keyframes cardExit {
  from {
    opacity: 1;
    transform: scale(1);
  }
  to {
    opacity: 0;
    transform: scale(0.95);
  }
}

duration: 200ms
easing: cubic-bezier(0.4, 0, 1, 1)
```

### 7.3 按钮点击反馈
```css
@keyframes buttonPress {
  0% { transform: scale(1); }
  50% { transform: scale(0.95); }
  100% { transform: scale(1); }
}

duration: 150ms
easing: ease-in-out
```

### 7.4 正确/错误反馈
```
认识（正确）：
- 卡片背景闪烁绿色（#7ED321, 20% opacity）
- 持续200ms

不认识（错误）：
- 卡片背景闪烁红色（#D0021B, 20% opacity）
- 持续200ms

跳过：
- 直接淡出，无额外反馈
```

## 8. 响应式设计

### 8.1 屏幕适配
虽然首版仅支持macOS，但需考虑不同分辨率：
- 标准分辨率（1920x1080）：默认尺寸
- 高分辨率（Retina）：使用@2x资源
- 低分辨率（1366x768）：卡片尺寸缩小10%

### 8.2 位置策略
```
默认位置：屏幕右上角
偏移量：
- 距离右边缘：40px
- 距离顶部：80px（避开菜单栏）

多显示器：
- 在主显示器上展示
- 不跨屏显示
```

## 9. 无障碍设计

### 9.1 键盘导航
- Tab键：在按钮间切换焦点
- Enter/Space：触发当前焦点按钮
- Esc：跳过当前卡片

### 9.2 焦点指示
```
焦点状态：
- Border: 2px solid #4A90E2
- Box-shadow: 0 0 0 3px rgba(74, 144, 226, 0.2)
```

### 9.3 对比度
所有文字与背景对比度 ≥ 4.5:1（WCAG AA标准）

## 12. 设计验证

### 12.1 可用性测试
1. 用户能否在3秒内完成一次学习？
2. 按钮点击区域是否足够大？
3. 文字是否清晰易读？
4. 动画是否流畅不卡顿？

### 12.2 视觉测试
1. 在不同分辨率下是否正常显示？
2. 在不同光线环境下是否易读？
3. 色彩搭配是否和谐？
4. 是否符合macOS设计规范？

### 12.3 性能测试
1. 动画帧率是否稳定在60fps？
2. 卡片渲染是否流畅？
3. 内存占用是否合理？
