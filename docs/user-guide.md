# 碎片时间背单词工具 用户文档

## 欢迎使用碎片时间背单词工具 👋

这是一款专为 macOS 设计的轻量级背单词应用，让你在工作间隙轻松记单词，无需专门安排学习时间。

## 快速开始

### 安装与启动

1. 下载并安装应用
2. 首次启动时，应用会自动：
   - 创建本地词库
   - 设置开机自启动
   - 常驻菜单栏

3. 在菜单栏找到应用图标（📚）

### 第一次使用

1. 点击菜单栏图标，查看今日统计
2. 等待 90 秒不操作电脑
3. 屏幕右上角会弹出第一张单词卡
4. 选择"认识"、"不认识"或"跳过"

就这么简单！

## 核心功能

### 1. 智能弹卡

**什么时候会弹出单词卡？**
- 当你停止操作电脑超过 90 秒
- 距离上次学习超过 25 分钟（兜底机制）

**不会打扰你的时候：**
- 夜间时段（23:00 - 07:00）
- 手动暂停期间
- 已有卡片显示时

### 2. 三种回答方式

#### 认识 ✅
- 快捷键：`Cmd + K`
- 表示：我记得这个单词的意思
- 结果：进入下一复习阶段，间隔时间延长

#### 不认识 ❌
- 快捷键：`Cmd + J`
- 表示：我不记得或记错了
- 结果：回退到更早的复习阶段，更快再次出现

#### 跳过 ⏭️
- 快捷键：`Esc`
- 表示：现在不方便，稍后再说
- 结果：不计入正确率，30分钟后可能再次出现

### 3. 复习系统

应用采用科学的间隔重复算法：

```
新词 → 10分钟 → 1天 → 3天 → 7天 → 14天 → 已掌握
```

- 答对：进入下一阶段
- 答错：回退一个阶段
- 连续答对：最终标记为"已掌握"

### 4. 菜单栏功能

点击菜单栏图标，可以：

**查看今日统计**
- 已学词数
- 正确率
- 新词数量
- 待复习词数

**控制学习节奏**
- ⏸️ 暂停 1 小时：临时关闭弹卡
- 🌙 今日不再提醒：今天不再弹出
- 📈 打开统计页：查看详细数据

**应用设置**
- ⚙️ 设置：调整学习参数
- ❌ 退出应用：完全退出

### 5. 统计页面

打开统计页可以看到：

**今日数据**
- 展示次数
- 认识次数
- 不认识次数
- 跳过次数
- 正确率
- 新学词数

**累计数据**
- 待复习词数
- 已掌握词数
- 总词库量
- 学习进度

## 使用技巧

### 最佳实践

1. **保持应用常驻**
   - 不要频繁退出应用
   - 让它在后台默默工作

2. **诚实作答**
   - 不确定就选"不认识"
   - 不要为了正确率而作弊

3. **合理使用跳过**
   - 真的很忙时再跳过
   - 不要养成习惯性跳过

4. **利用快捷键**
   - `Cmd + K`：认识
   - `Cmd + J`：不认识
   - `Esc`：跳过
   - 比鼠标点击更快

### 学习节奏建议

**每日新词量**
- 默认：15 个新词/天
- 建议：根据自己的时间调整
- 原则：宁少勿多，保证质量

**复习优先**
- 系统会优先展示到期复习词
- 新词在复习词之后
- 确保旧词不遗忘

**暂停使用**
- 开会前：暂停 1 小时
- 专注工作：今日不再提醒
- 晚上休息：自动进入夜间静默

## 常见问题

### Q: 为什么没有弹出单词卡？

**可能原因：**
1. 没有满足 90 秒 idle 条件
2. 当前在夜间静默时段（23:00-07:00）
3. 已手动暂停
4. 今日新词配额已用完且无待复习词

**解决方法：**
- 检查菜单栏是否显示"暂停"状态
- 确认当前时间不在静默时段
- 查看统计页确认是否有待学习词

### Q: 单词卡自动消失了怎么办？

单词卡默认显示 10-12 秒后自动消失，这是正常行为。

- 自动消失会被记录为"跳过"
- 不影响学习进度
- 下次会再次出现

### Q: 如何调整弹卡频率？

首版暂不支持自定义弹卡频率，默认策略：
- Idle 触发：90 秒
- 兜底触发：25 分钟

后续版本会支持自定义。

### Q: 可以导入自己的词库吗？

首版仅支持内置的雅思核心 3000 词。

后续版本会支持：
- 自定义词库导入
- 多词库切换
- 词库编辑

### Q: 数据存储在哪里？

所有数据存储在本地 SQLite 数据库：
- 位置：`~/Library/Application Support/fragment-vocab/`
- 包含：词库、学习记录、统计数据
- 不会上传到云端

### Q: 如何备份学习数据？

手动备份方法：
1. 退出应用
2. 复制数据库文件：
   ```
   ~/Library/Application Support/fragment-vocab/vocab.db
   ```
3. 保存到安全位置

恢复方法：
1. 退出应用
2. 替换数据库文件
3. 重新启动应用

### Q: 会影响电脑性能吗？

不会。应用非常轻量：
- 内存占用：< 50MB
- CPU 占用：几乎为 0（idle 时）
- 磁盘占用：< 10MB

### Q: 支持多设备同步吗？

首版不支持。

后续版本可能支持：
- iCloud 同步
- 手动导出/导入
- 跨设备学习进度同步

## 学习建议

### 适合的使用场景

✅ **推荐：**
- 工作间隙的短暂休息
- 等待编译/构建的时间
- 喝水、上厕所回来后
- 思考问题的间隙

❌ **不推荐：**
- 正在开会或通话
- 需要高度专注的工作
- 正在演示或分享屏幕
- 深夜或早晨刚起床

### 学习目标设定

**短期目标（1-2周）**
- 熟悉应用操作
- 建立学习习惯
- 每天完成 10+ 次学习

**中期目标（1-3个月）**
- 掌握 500+ 核心词汇
- 保持 70%+ 正确率
- 形成自然的学习节奏

**长期目标（3-6个月）**
- 掌握 1500+ 雅思词汇
- 正确率稳定在 80%+
- 为雅思考试打下基础

### 配合其他学习方式

本应用适合作为：
- **碎片时间补充**：利用工作间隙
- **词汇量积累**：快速扩充词汇
- **复习巩固**：科学间隔重复

建议配合：
- 系统的雅思课程
- 阅读英文文章
- 观看英文视频
- 实际使用场景练习
