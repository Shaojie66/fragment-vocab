# Fragment Vocab 用户指南

## 产品形态

当前版本的 Fragment Vocab 由 3 个部分组成：

1. 主页面  
设置控制台。负责查看状态、调整策略、完成首次设置、导入导出配置。

2. 托盘  
辅助入口。负责暂停、今日不再提醒、打开主页面或统计页。

3. 浮卡  
实际作答界面。负责认识 / 不认识 / 跳过，以及“这张词先别再推”的轻反馈。

## 首次启动

第一次打开应用时，会先进入最小设置引导。需要确认：

- 每日新词目标
- 提醒偏好
- 静默开始 / 结束时间
- 是否开机启动

完成后，系统才会开始正常调度。

## 主页面能做什么

主页面不是简单统计页，而是设置控制台。当前可直接完成：

- 查看今日学习状态
- 查看当前推荐模式和推荐原因
- 修改提醒模式、idle 阈值、兜底频率
- 修改静默时间、工作日 / 周末策略
- 修改每日新词上限和学习计划
- 修改卡片偏好
- 修改启动行为、托盘和开机启动
- 暂停 1 小时 / 今日不再提醒 / 恢复提醒
- 应用团队模板
- 记录“提醒太多 / 太少”反馈
- 导出配置摘要 / JSON
- 从 JSON 导入配置

## 浮卡怎么工作

浮卡只在满足条件时弹出。默认逻辑是：

- 优先在空闲达到阈值后触发
- 如果很久没有弹出，会按兜底频率补一次
- 夜间静默、暂停、无卡可学时不会触发

浮卡支持：

- 认识
- 不认识
- 跳过
- 这张词先别再推

说明：

- “跳过”不会推进学习进度，但会进入冷却
- “这张词先别再推”会记录一条反馈，同时按跳过处理

## 当前推荐和自定义

主页面里的提醒策略分成两层：

1. 系统推荐  
系统会根据默认策略，以及最近的跳过率、暂停状态和反馈，给出推荐说明。

2. 用户自定义  
你可以切到自定义模式，直接覆盖 idle 阈值和兜底频率。  
系统仍然会显示建议，但不会自动覆盖你的设置。

## 工作日与周末策略

如果你使用推荐模式，系统会按日期自动切换：

- 工作日推荐策略
- 周末推荐策略

这两个配置只影响推荐层；如果你切到自定义，则以自定义参数为准。

## 统计页

统计页当前会展示：

- 今日学习次数
- 正确率
- 今日新词
- 待复习词
- 认识 / 不认识 / 跳过次数
- 已掌握词数
- 当前推荐说明
- 当前暂停 / 配置摘要
- 最近反馈

统计页也可以直接跳回主页面。

## 常用操作

### 暂停提醒

- 主页面点击“暂停 1 小时”
- 或托盘里选择“暂停 1 小时”

### 今日不再提醒

- 主页面点击“今日不再提醒”
- 或托盘里选择“今日不再提醒”

### 恢复提醒

- 主页面点击“恢复提醒”

### 打开统计页

- 主页面点击“打开统计页”
- 或托盘中打开

### 导出配置

主页面“导出与分享”区域支持：

- 生成文字摘要
- 复制摘要
- 复制 JSON
- 下载 JSON

### 导入配置

主页面“导出与分享”区域支持从之前导出的 JSON 导入。

## 当前仍不支持

- 自定义词库导入
- 多设备同步
- 自定义浮卡位置
- 自定义快捷键

## 本地数据

应用数据保存在本地目录：

- `~/Library/Application Support/com.chenshaojie.fragment-vocab/fragment-vocab.db`

如果要手动备份：

1. 退出应用
2. 复制上面的数据库文件
3. 保存到安全位置

## 建议的试用方式

1. 先完成首次引导，不要直接保持默认值
2. 先用 1-2 天观察推荐是否偏紧或偏松
3. 如果不适合，再切到自定义
4. 团队讨论时优先使用导出摘要或 JSON 分享配置
