# Fragment Vocab FAQ

## 基础使用

### Q1: 现在还是纯托盘应用吗？
不是。当前版本会以主页面作为产品主入口，托盘只作为辅助入口。

### Q2: 打开应用后应该看到什么？
首次使用会先看到最小设置引导；完成后进入主页面设置控制台。  
如果你直接双击应用却没看到窗口，先确认是否已有实例在后台运行。

### Q3: 浮卡什么时候会弹出？
满足以下条件时才会弹：

- 当前未暂停
- 不在静默时段
- 当前没有其他浮卡显示
- 有可学习卡片
- 空闲时间达到阈值，或触发兜底频率

### Q4: 为什么今天没有弹卡？
常见原因：

- 当前处于暂停状态
- 当前处于夜间静默
- 当前没有可学习卡片
- 空闲时间还没达到阈值

主页面的“解释与诊断”区会显示最近阻塞原因。

## 设置与推荐

### Q5: 可以改提醒频率吗？
可以。现在支持：

- 使用系统推荐
- 切到自定义模式手动改参数
- 一键恢复推荐

### Q6: 系统推荐会不会强行覆盖我的设置？
不会。系统只给建议和解释；如果你已经切到自定义，当前运行参数仍以你的设置为准。

### Q7: 可以改静默时间吗？
可以。主页面“时间与静默”区域可直接调整开始和结束时间。

### Q8: 可以区分工作日和周末吗？
可以。推荐模式下支持分别配置工作日和周末的策略。

### Q9: 可以调整每日新词数量吗？
可以。主页面“学习计划”区域支持直接修改每日新词上限。

### Q10: 可以改卡片展示偏好吗？
可以。当前支持：

- 自动隐藏时间
- 英文优先 / 中文优先
- 是否显示音标
- 是否允许跳过
- 是否启用快捷键

### Q11: 可以改快捷键本身吗？
当前还不支持改键位，只支持打开或关闭快捷键。

## 托盘、统计与反馈

### Q12: 托盘现在还有什么用？
托盘只保留辅助能力：

- 打开主页面
- 打开统计页
- 暂停 1 小时
- 今日不再提醒
- 退出应用

### Q13: 统计页现在展示什么？
统计页当前会展示：

- 今日学习指标
- 当前推荐说明
- 暂停与配置摘要
- 最近反馈

### Q14: 有应用内反馈入口吗？
有。现在支持：

- 主页面反馈“提醒太多 / 太少”
- 浮卡反馈“这张词先别再推”

### Q15: 可以导出配置吗？
可以。主页面支持：

- 生成摘要
- 复制摘要
- 复制 JSON
- 下载 JSON

### Q16: 可以导入配置吗？
可以。主页面支持从之前导出的 JSON 重新导入配置。

## 数据与同步

### Q17: 数据保存在哪里？
保存在本地：

- `~/Library/Application Support/com.chenshaojie.fragment-vocab/fragment-vocab.db`

### Q18: 会上传云端吗？
不会。当前版本仍然是本地优先，不做云同步。

### Q19: 支持多设备同步吗？
当前不支持。

## 词库

### Q20: 当前内置什么词库？
当前默认导入内置的 IELTS Core 3000 词库。

### Q21: 可以导入自己的词库吗？
当前还不支持自定义词库导入。

## 已知边界

### Q22: 可以自定义浮卡位置吗？
当前还不支持。

### Q23: 可以自定义快捷键吗？
当前还不支持。

### Q24: 可以生成更丰富的学习报告吗？
当前只支持配置摘要和 JSON 导出，不支持 PDF/CSV 学习报告。
